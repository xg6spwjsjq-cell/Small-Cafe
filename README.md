# Cafe Order Management System

## Description
A simple console application designed for a small cafe. This project demonstrates basic Object-Oriented Programming (OOP) principles in Java, including the use of classes, fields, constructors, static members, `ArrayList`, and method overriding (`toString` and `equals`). The system allows the creation of menu items, registration of customers, and building individual orders with total cost calculation.

## Class List
1. **MenuItem**: Represents a single product sold in the cafe. Tracks the total number of products created.
2. **CafeCustomer**: Represents a customer who places an order.
3. **Order**: Represents a customer's order containing a list of `MenuItem` objects, handles total calculation, and manages automatic order numbering.
4. **Main**: Contains the test scenario demonstrating the program flow.
